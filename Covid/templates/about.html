Summary

Currently, we are unable to determine the next location of the spread of the COVID-19 virus. However, we can predict the probability of where it would spread next. Our project aims to develop a COVID-19 hotspot prediction system using environmental variables such as air quality, meteorological variables, and human population. Based on our research, the air quality and the weather had a positive correlation with spread of the virus. Users will be able to see which city is safe during pandemic.

How it works?

The user will input the environmental variables and our prediction model will output a value of how many cases will the virus produce. The higher the number of cases, the more cautious the user should be in the chosen city.

Random Forest Regressor is used to predict the number of COVID-19 cases.

Tools used:

Python for Web Design and Analysis

Flask, Sklearn, Numpy, Matplotlib

Docker and Azure for production
